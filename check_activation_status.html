Check Activation Status

You can check the result after activating with KinovRise Activator as follows:​

info

1. Check right in the tool​

  • You can check the activation status after the activation process is completed, if Windows or Office has been activated, the system will notify that
Windows/Office has already activated!

2. Check using Settings in Windows​

You can check again to see if Windows is activated by going to Settings

  • Step 1: Go to Windows Settings
  • Step 2: Type in the search bar " See if Windows is Activated "
  • Step 3: If you see " Windows is activated using your organization's activation service " then congratulations you have successfully activated with KRA!
